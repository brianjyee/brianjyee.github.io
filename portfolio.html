Coderaising


Coderaising

Coderaising is a 4-weekend hackathon aimed at bringing mentors and apprentice programmers together to work on non-profit, open source projects. I was a participant in the pilot class of this program. My team's project was to rebuild and improve upon the existing Coderaising website. My team built functionality that allows mentors to create projects, users to apply to be part of a team, and teams to collaborate and work together. The site is built with Python/Django.

Sports News and Odds

This is a sports news site I built using the ESPN api. It displays the latest sports news stories. It also has Vegas odds for upcoming sporting events. This is still a work in progress. The site is built with Python/Django.
